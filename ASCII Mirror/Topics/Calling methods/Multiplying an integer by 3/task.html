Write a Java program that includes a method which multiplies any given integer by 3. The main method should take an integer as input from the user, call the multiplication method with the input integer as the argument and print the result. Input: A single line containing an integer 'n'(-10^3 <= n <= 10^3). Output: A single line containing the result of the integer 'n' multiplied by 3.
Sample Input:
5

Sample Output:
15


Sample Input:
-6

Sample Output:
-18



Memory limit: 256 MB
Time limit: 5 seconds
