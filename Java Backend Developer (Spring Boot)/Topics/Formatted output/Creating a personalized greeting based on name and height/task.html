You are given a string which represents the name of a person and an int which represents the person's height in centimeters. Your task is to create a java program that can take these inputs and print a statement like this: "Hello, my name is (name) and I am (height) cm tall.". Your program should accept two inputs: a String and an int, in this order. The first input is the name of the person, and the second input is the person's height.


Sample Input:
John
180

Sample Output:
Hello, my name is John and I am 180 cm tall.


Sample Input:
Alice
155

Sample Output:
Hello, my name is Alice and I am 155 cm tall.



Memory limit: 256 MB
Time limit: 5 seconds
