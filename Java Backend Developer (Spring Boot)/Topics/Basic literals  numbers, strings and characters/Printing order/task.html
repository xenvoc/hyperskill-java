Printing order

Reading code is crucial for anyone in IT, so let's practice this valuable skill together. Consider the program below and determine the order in which it will print the literals:

public class Main {
    public static void main(String[] args) {
        int exmpl = 1_000_000;
        String line = "A";
        System.out.println('A');
        System.out.println(line);
        System.out.println(exmpl);
    }
}
